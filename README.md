## Description

User recommendation api using [Nest](https://github.com/nestjs/nest) framework and TypeScript.

## Running the api

```bash
$ docker-compose up -d
```

## Swagger Docs
http://localhost:3000/api

## Running test cases
#### install dependencies
```bash
$ npm install
```
#### unit tests
```bash
$ npm run test
```
#### end to end tests
```bash
$ npm run test:e2e
```
